How it works

One page, three moves

01

Capture

Click the extension on any live page. It freezes the real DOM and real CSS into one standalone file — no auth, no live JS.

02

Prototype

Describe the change in plain language. Claude edits the markup directly, in place, with full context of what's already there.

03

Export

Pull a clean diff for handoff, or download the finished HTML on its own — ready to share or build from.

Get started

Three steps, one time

1

Download & unzip

Use the button above, or grab it straight from the repo.

github.com/liorA2102/page-bender
2

Double-click setup.command

Installs the local server, checks you're logged into Claude Code, and opens the next step for you.

./setup.command
3

Load it into Chrome

In the Chrome tab that just opened: turn on Developer Mode, click "Load unpacked," and pick the folder that's already highlighted in Finder.

i

Before you install

The local server runs with full tool access on your machine — the editing agent can run shell commands while it works, with no sandbox. That's a deliberate trade-off for how capable the editing feels, not an oversight. Only install this on a machine you're comfortable with that.
